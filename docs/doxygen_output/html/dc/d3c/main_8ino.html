◆ lerDistanciaMM()

int lerDistanciaMM ( uint8_t  pinTrig,
uint8_t  pinEcho 
)

Lê a distância em milímetros usando sensor ultrassônico HC-SR04.

Parâmetros
pinTrig- Pino de trigger (disparo do pulso)
pinEcho- Pino de echo (recepção da reflexão)
Retorna
Distância em milímetros (sem uso de float para economizar memória)

Fórmula: (Duração * 10) / 58 = Distância em mm (Equivalente a dividir por 5.8 usando apenas operações inteiras)

◆ processarEnchimento()

void processarEnchimento ( NexProgressBar &  barra,
uint8_t  pinoMotor 
)

Processa o enchimento de bebida com monitor de copo.

Responsabilidades:

  • Controla o motor da bomba
  • Incrementa a barra de progresso
  • Detecta remoção do copo durante o processo
  • Pausa e retoma automaticamente se copo for recolocado
Parâmetros
barra- Referência à barra de progresso Nextion
pinoMotor- Pino do relé do motor (CAFE ou CHA)
